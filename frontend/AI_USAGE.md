# AI Usage Statement

This project was developed primarily by me as part of my frontend developer portfolio.

During development I used **AI tools (ChatGPT)** to assist with certain tasks such as:

- Writing and improving code comments
- Suggesting commit message wording
- Improving typography and documentation text
- Explaining concepts related to Next.js, Sanity CMS and TypeScript
- Helping structure project architecture

## Important clarification

AI tools were used **as assistants**, not as autonomous code generators.

All code in this repository was:

- reviewed
- understood
- integrated
- and adapted

by me.

The final architecture, implementation decisions and debugging were performed by me.

## Why disclose AI usage?

Modern development workflows often include AI tools.  
This document exists to maintain transparency about how AI was used in this project.

The goal is to demonstrate **honest development practices and clear attribution of assistance tools**.
